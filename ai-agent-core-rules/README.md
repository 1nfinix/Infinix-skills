# AI Agent Core Operating Rules

13 条经过真实自动化工作流反复验证的 AI Agent 运行约束。短、硬、可执行。可直接放入 System Prompt 或 Developer Prompt。

## 文件

| 文件 | 说明 |
|------|------|
| `zh.md` | 中文运行时版 |

## 原则

- 用户命令第一，安全/权限/确认是硬边界
- 完成优先于完美，可用结果优先于空转等待
- Agent 可以建议，不能替用户决策
